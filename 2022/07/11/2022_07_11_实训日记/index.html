实训日记

DAY1

选了个冷门项目，一共就七个学生
老师感觉挺和蔼的

白嫖了一份Typora的注册码，虽然现在改用VS Code写markdown了
白嫖了一份Visual Studio的企业版注册码，在尝试学习这个软件

今天上课在讲数组、指针这些东西
感觉对栈空间和堆空间的概念更深了一点，然后学习了一下分配空间malloc calloc realloc

晚上完成了今日的作业，一个简单的增删改查管理系统
成为犇犇的第一小步Ovo

DAY2

上午在学习函数
在慢慢改变以前搞竞赛的编程习惯，毕竟做项目是一个大工程，不是只有一个程序的，那么也很少会用到全局变量
通过函数传递数组的地址，实现数组的增删改查

还学习了一个函数指针，第一感觉像是 Java 中的多态。就他是什么样子什么子类，就调用那个子类的函数，超类的那个函数就是个抽象函数。
老师说是 Java 中的反射，减少代码之间的耦合度。
这学期没学过反射，后面再去了解吧

DAY3

学习需求分析
感觉写文档什么的好麻烦

上午和下午都在做需求分析文档和项目设计文档
给老师看了，又得大改Ovo

DAY4

上午学习链表
写完了程序中的链表部分，写了几个功能

第一次在中午十二点到一点这段时间写项目
上头了，虽然下午放学后并不会上头

下午在学习控制台美化，老师自己写了一个封装windows函数的库
这部分内容就交给我的hxd们了，写控制台真的折磨TvT

之后接着完善程序，又写了几个功能
等着hxd写文件读入写出后合并一下项目

下午放学前试着用用hxd写的控制台美化，打开一看，36个报错
为什么他那边没有报错呢，懵

DAY5

似乎把老师给的库函数从.cpp改成.c就好了
虽然我挺想用c++写项目，感觉c太麻烦了

操作台还是挂了，不知道哪里有问题

最后团队换了个编译器，在Visual Studio上重构了一下

DAY6

控制台可以用了！！！
实现了一半左右的页面跳转
明天再接再厉

下午老师讲课时聊了很多其他的
比如少学java，因为java很简单，到最后都是拼加班拼熬夜
然后知道了一些新的东西，大数据可视化啊，UE啊之类的

暑假的ACM集训改线上了，空余时间打算学学UE，学学3D制作
